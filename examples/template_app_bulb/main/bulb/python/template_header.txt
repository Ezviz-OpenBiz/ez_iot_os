property_$(key)_up,   设备主动上报的属性值调用此函数
property_$(KEY)_set,  sdk 底层接收到的属性协议处理函数
action_$(KEY)_set，    sdk 底层接收到的操作协议处理函数 
event_$(KEY)_up     设备主动上报事件调用此函数

可以通过python 脚本将控制台下载的物模型文件转化为此文件
*/
